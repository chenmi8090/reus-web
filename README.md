## 介绍

Reus-Web是一个基于Reus代码生成项目的前端页面,采用的是Vue技术,基于Vue-Admin-Template模板进行设计。
因为项目是一个小项目所以是一体式架构,所以需要打包前端代码与后台进行整合后运行。
